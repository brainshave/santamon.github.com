Links

Any word ending with .html pointing to existing file or to an html file that has its *.txt source or beginning with an http(s):// will treated as a link. The word immediately before address will be text of the link. If it's ! then link address will be used.

Tests:

  • http://pstryk.net/~santamon sdfs sadf
  • UIOP sdfwefw
  • ASDF wefwe
  • qwer uiop
  • tyuio ghjk tyui
  • sd fas Syntax.html wefwe
  • tyu erty erty wert wefwe
  • QWER http://pstryk.net/~santamon wfewe we
  • asdf asdfas

Also, appending ! to the address will make it plain text.

Images

All words ending with .png, .jpg, .gif, .bmpasdfF (maybe some more) will be translated to inlined thumbnail pointing to original image, only if they point to existing file or if they're begining with http:// or https://. If ! is added like in asdf.png! , original image will be inlined.

Positioning

Paragraphs that begins with be right-aligned, right-floated, with auto width.

Analogical to =>.

Paragraphs that begins with -> will be right-aligned. asdf

Paragraphs that begins with | will be centered.
